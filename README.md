# FNB App of the Year: Python Course (Units 1–6)

This repository contains my practical projects, code exercises, and notes from the **FNB App of the Year Python Course**. The curriculum covers core programming principles, foundational syntax, logic structures, and advanced data manipulation to build production-ready applications.

## Course Overview
The course transitions students from fundamental programming syntax to advanced object-oriented design and file automation. It focuses heavily on clean code, algorithm development, and logical problem-solving using Python.

---

## Curriculum & Topics Covered

### Unit 1: Programming Basics & Environment Setup
* **Introduction to Python:** Understanding interpreted languages and setup.
* **Development Environment:** Configuring IDEs and running scripts.
* **Basic Syntax:** Printing output and basic string manipulation.
* **Comments:** Writing clean, documented code.

### Unit 2: Variables, Data Types & Operators
* **Variables:** Dynamic typing, memory allocation, and naming conventions.
* **Data Types:** Working with Strings, Integers, Floats, and Booleans.
* **Type Casting:** Converting data between different types.
* **Operators:** Mastering arithmetic, assignment, and comparison operators.

### Unit 3: Conditional Logic & Control Flow
* **Boolean Logic:** Evaluating statements using `AND`, `OR`, and `NOT`.
* **If-Elif-Else Structures:** Building decision-making mechanisms.
* **Nested Conditionals:** Handling multi-layered logical scenarios.
* **Error Handling Basics:** Preventing crashes with defensive logic.

### Unit 4: Loops & Iterations
* **While Loops:** Executing code blocks based on conditional states.
* **For Loops:** Iterating over sequences, ranges, and collections.
* **Loop Control:** Utilizing `break`, `continue`, and `pass` statements.
* **Nested Loops:** Managing multi-dimensional data iterations.

### Unit 5: Data Structures (Collections)
* **Lists:** Creating, indexing, slicing, and mutating ordered arrays.
* **Tuples:** Utilizing immutable sequences for fixed data sets.
* **Dictionaries:** Managing key-value pairs for optimized data lookups.
* **Sets:** Handling unordered collections of unique elements.

### Unit 6: Functions & Modular Programming
* **Function Definition:** Declaring reusable code blocks using `def`.
* **Arguments & Parameters:** Passing positional, keyword, and default values.
* **Return Statements:** Outputting data and state results from functions.
* **Scope:** Understanding global vs. local variable accessibility.
